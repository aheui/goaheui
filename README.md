# GoAheui

[Aheui](http://aheui.github.io/specification.en) implementation in Go.

Aheui is an esoteric programming language using the Korean alphabet.

## Install

    go get github.com/sungwoncho/goaheui

## Usage

    $ aheui path_to_file

## Hello world
